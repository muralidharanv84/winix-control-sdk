# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Standards

Examples of behavior that contributes to a positive environment include:

- Using welcoming and inclusive language
- Respecting differing viewpoints and experiences
- Giving and receiving constructive feedback
- Focusing on what is best for the community

Examples of unacceptable behavior include harassment, insults, and personal attacks.

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior and may take corrective action for unacceptable behavior.

## Reporting

Report issues to the repository maintainers through GitHub issues or direct contact.

This Code of Conduct is adapted from the Contributor Covenant, version 2.1.
